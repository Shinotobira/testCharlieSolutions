## Tech Stack

**Client:** Dart, Flutter

**DataBase:** Sqlite


## Run Locally

Clone the project

```bash
  git clone https://github.com/Shinotobira/testCharlieSolutions
```

Go to the project directory

```bash
  cd testCharlieSolutions
```

Install dependencies

```bash
  flutter pub get
  flutter channel master
```

Launch the app

```bash
  flutter run main.dart
```
